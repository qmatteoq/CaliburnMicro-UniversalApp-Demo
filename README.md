# Caliburn.Micro 2.0 — Universal Windows app samples

A set of samples that show how to use **Caliburn.Micro 2.0** in Universal Windows
Platform (UWP) applications: conventions-based binding, actions, navigation and the
MVVM building blocks the framework provides.

> **Historical sample.** These projects target UWP on Windows 10 and Caliburn.Micro 2.0.
> They are kept as a reference; new apps should evaluate current framework versions and
> **WinUI 3 / Windows App SDK**.

## What's included

| Folder | What it demonstrates |
| --- | --- |
| `Actions` | Wiring UI events to view-model methods using Caliburn.Micro's action/convention system. |
| Additional folders | Further Caliburn.Micro scenarios (binding conventions, navigation, etc.). |

Each sample includes the shared project plus the Windows and Windows Phone heads.

## Prerequisites

- Windows 10
- Visual Studio 2015 with the **Universal Windows Platform development** workload
- Windows 10 SDK

## Getting started

1. Clone the repository.
2. Open the `.sln` inside the sample folder you want to run.
3. Restore NuGet packages and run (F5).
